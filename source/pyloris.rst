==========
PyLoris
==========

PyLoris 是用于测试服务器漏洞，进行连接耗尽性能拒绝服务的 DoS 攻击可脚本工具。
PyLoris 可以利用 SOCKS 代理和 SSL 连接，并且可以指定协议，如：HTTP、FTP、SMTP、IMAP、Telnet。

* 网站：https://motoma.io/pyloris/
* SourceForge：https://sourceforge.net/projects/pyloris/

特性
==========

* Tkinter GUI
* Scripting API
* Anonymity
* TOR Proxying
* SOCKS Proxying
